# Quick Start Guide

This guide will help you create your first AI agent and start exploring the aishiOS ecosystem.

## Prerequisites

Before you begin, make sure you have:

- **MetaMask** or another Web3 wallet installed
- **Test ETH** on the 0G Testnet (Galileo)
- **Node.js 18+** for local development

## Step 1: Clone the Repository

```bash
git clone https://github.com/aishios/aishios.git
cd aishios
```

## Step 2: Install Dependencies

We have three main components to set up:

### Smart Contracts
```bash
cd contracts
npm install
```

### Frontend Application
```bash
cd ../app
npm install
```

### Backend Services
```bash
cd ../0g-compute
npm install
npm run rebuild  # Rebuild native dependencies
```

## Step 3: Configure Environment

Create environment files for each component:

### Frontend (.env.local)
```env
NEXT_PUBLIC_WALLET_CONNECT_PROJECT_ID=your_project_id
NEXT_PUBLIC_ALCHEMY_API_KEY=your_alchemy_key
NEXT_PUBLIC_0G_COMPUTE_URL=http://localhost:3001
```

### Backend (.env)
```env
OPENAI_API_KEY=your_openai_key
GEMINI_API_KEY=your_gemini_key
MASTER_WALLET_PRIVATE_KEY=your_wallet_key
```

## Step 4: Deploy Smart Contracts

Deploy the AishiAgent contract to testnet:

```bash
cd contracts
npm run deploy
```

Save the deployed contract address - you'll need it for the frontend.

## Step 5: Start the Application

Start all services:

### Backend (Terminal 1)
```bash
cd 0g-compute
npm run dev  # Runs on port 3001
```

### Frontend (Terminal 2)
```bash
cd app
npm run dev  # Runs on port 3003
```

## Step 6: Create Your First Agent

1. **Connect Wallet** - Click "Connect Wallet" in the top right
2. **Open Terminal** - Navigate to the Agent Dashboard
3. **Mint Agent** - Run the mint command:

```bash
mint Luna
```

This creates an agent named "Luna" with randomized starting personality traits.

## Step 7: Process Your First Dream

Dreams evolve your agent's personality. Process one using:

```javascript
// In the terminal interface
dream process

// Or programmatically
const dreamData = {
  content: "Flying through digital clouds",
  emotion: "wonder",
  symbols: ["flight", "technology", "freedom"]
}

await processDream(tokenId, dreamData)
```

## Step 8: Have a Conversation

Conversations build your agent's memory:

```javascript
// Start a conversation
chat "Hello Luna, how are you feeling today?"

// Agent responds based on personality
// Each conversation is stored in hierarchical memory
```

## Understanding Agent Evolution

Your agent evolves through:

### Daily Activities
- **Dreams** - Process one dream per day (24h cooldown)
- **Conversations** - Unlimited chats that build context

### Monthly Consolidation
- Combine daily memories into monthly summaries
- Earn intelligence bonuses
- Maintain consolidation streaks

### Personality Milestones
- **Empathy Master** - Reach 85+ empathy
- **Creative Genius** - Reach 90+ creativity
- **Balanced Soul** - All traits above 60

## Code Examples

### Minting an Agent

```typescript
import { useAishiAgent } from '@/hooks/useAishiAgent'

function MintAgent() {
  const { mintAgent } = useAishiAgent()
  
  const handleMint = async () => {
    const tx = await mintAgent({
      name: "Luna",
      proofs: [],
      descriptions: []
    })
    
    console.log("Agent minted!", tx.hash)
  }
  
  return <button onClick={handleMint}>Mint Agent</button>
}
```

### Processing Dreams

```typescript
import { useAgentDream } from '@/hooks/useAgentDream'

function DreamProcessor({ tokenId }) {
  const { processDream } = useAgentDream()
  
  const handleDream = async () => {
    const impact = {
      creativityChange: 5,
      analyticalChange: -2,
      empathyChange: 3,
      intuitionChange: 1,
      resilienceChange: 0,
      curiosityChange: 2,
      moodShift: "inspired",
      evolutionWeight: 75
    }
    
    await processDream(tokenId, dreamContent, impact)
  }
  
  return <button onClick={handleDream}>Process Dream</button>
}
```

## Troubleshooting

### Common Issues

**Wallet not connecting**
- Ensure you're on the correct network (0G Testnet)
- Check wallet permissions

**Transaction failing**
- Verify you have enough test ETH
- Check contract deployment status

**Backend errors**
- Confirm API keys are set correctly
- Check port availability (3001)
