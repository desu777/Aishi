    getSigningAddress

    • getSigningAddress(
          providerAddress: string,
          verifyRA?: boolean,
      ): Promise<InferenceSingerRAVerificationResult>

      getSigningAddress verifies whether the signing address of the signer corresponds to a valid RA.

      It also stores the signing address of the RA in localStorage and returns it.

      Parameters

      • providerAddress: string

        provider address.

      • verifyRA: boolean = false

        whether to verify the RA， default is false.

      Returns Promise<InferenceSingerRAVerificationResult>

      The first return value indicates whether the RA is valid, and the second return value indicates the signing address of the RA.
